Kimberly and Mackenzie's Dodge the Falling Platforms Game

Description

Our game is a simple brick breaker style game where the player's goal is to destroy all of the bricks. The player controls the platform on the bottom by using the 'A' and 'D' keys to move left and right respectively. Using the platform to bounce the ball, the player must destroy every brick without letting the ball fall past the platform.


Reflection

Personally, I think the game turned out pretty well with all the basic functions we had planned, but I wish we had more time to add extra functionalities such as power-ups. The hardest part of making this app was programming the blocks to move with two second increments between each other. Since the MIT App Inventor doesn't have a 'wait' function, we couldn't easily program the blocks within a loop. We ended up having to use five separate clocks, one for each block, to time the start of their movement.
